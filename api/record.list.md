# record.list（查询收支记录）
- 请求方式：GET
- 请求参数：

| 字段名   | 中文名称 | 非空 | 长度 | 备注 |
|timeStart| 起始时间 | Y   |      | yyyy-MM-dd HH:mm:ss|
|timeEnd  | 结束时间 | N   |      | yyyy-MM-dd HH:mm:ss，默认当前时间|
